# منصة آيات — توصيف الهوية البصرية وتجربة الاستخدام

يقدّم هذا المستند تصورًا شاملًا لواجهة منصة «آيات» المتخصصة في تحفيظ القرآن الكريم وتعليم التجويد عن بُعد، مع التركيز على الهوية البصرية، الصفحات الرئيسة، ولوحات التحكم الخاصة بمختلف أنواع المستخدمين.

## 1. الهوية البصرية (Branding)
- **الألوان الأساسية**: أخضر رئيسي `#16A34A` يعكس الطابع القرآني والطمأنينة، مع رمادي فاتح `#F3F4F6` للخلفيات، وألوان نصية بدرجات الأسود أو الرمادي الداكن لسهولة القراءة.
- **الخطوط**: استخدام خط عربي واضح وحديث مثل «Cairo» أو «Tahoma»، مع دعم الخطوط اللاتينية المقابلة عند عرض الواجهة بالإنجليزية.
- **الأسلوب البصري**: تصميم بسيط (Minimal) يتضمن لمسة إسلامية هادئة عبر الأيقونات والزخارف الخفيفة، مع الحفاظ على تباين واضح ومساحات بيضاء مريحة.
- **الاتجاه**: دعم كامل لاتجاه الكتابة من اليمين إلى اليسار (RTL) مع إمكانية التحويل للإنجليزية (LTR) عند الحاجة.
- **الوضعيات**: توفير وضعي العرض الفاتح والداكن مع الحفاظ على الهوية اللونية.

## 2. الصفحة الرئيسية (Landing Page)
### الهيدر
- شعار «آيات» مثبت في الزاوية اليمنى.
- زر «تسجيل الدخول» في الزاوية اليسرى دون قوائم معقدة لعدم تشتيت الزائر.

### قسم الترحيب (Hero Section)
- عنوان رئيسي بارز: «آيات — منصة تحفيظ القرآن وتعليم التجويد عن بُعد».
- وصف مختصر يشرح القيمة المضافة للمنصة.
- زران أساسيان: «ابدأ الآن» و«تسجيل الدخول».

### قسم المميزات (Features)
- شبكة من 6 بطاقات تبرز: إشراف الشيخ، مصحف تفاعلي، نظام الحضور والغياب، دفع آمن، دورات متنوعة، تقارير متابعة.

### قسم الدورات (Courses)
- أربع بطاقات للدورات الرئيسة: الحفظ، التجويد، المراجعة، الإجازة، مع زر «احجز الآن» لكل بطاقة.

### قسم الأسعار (Pricing)
- ثلاث باقات: شهرية، دورات، مميزة. كل باقة تحتوي على السعر، وصف مختصر، وزر «اشترك».

### الفوتر
- خلفية باللون الرمادي الفاتح.
- نص الحقوق: «© 2025 منصة آيات».
- روابط سفلية صغيرة: «سياسة الخصوصية» و«تواصل معنا».

## 3. صفحات الحساب
### تسجيل الدخول
- نموذج يضم حقول البريد الإلكتروني، كلمة المرور، وزر «دخول».
- رابط نصي لإنشاء حساب جديد.

### إنشاء حساب
- الحقول: الاسم الكامل، البريد الإلكتروني، كلمة المرور، اختيار الدور (طالب / شيخ)، وزر «تسجيل».

## 4. لوحة الطالب (Student Dashboard)
- قائمة جانبية تتضمن: الحصص، الحضور، التقدّم، الدورات.
- المحتوى الرئيس يشمل جدول الحصص القادمة، تقارير الحفظ (نسبة مئوية + ملاحظات الشيخ)، وأيقونة إشعارات (جرس) في أعلى الصفحة.

## 5. لوحة الشيخ (Teacher Dashboard)
- قائمة جانبية لعناصر: الطلاب، جدول الحصص، تسجيل الحضور، متابعة الحفظ.
- واجهة تعرض قائمة الطلاب وحالة كل طالب، إمكانية فتح تقارير فردية، وأزرار لإضافة الملاحظات أو تسجيل الغياب.

## 6. لوحة المدير (Admin Dashboard)
- قائمة جانبية لعناصر: إدارة المستخدمين، إدارة الدورات، المواعيد، المدفوعات، الإعدادات.
- جداول بيانات بسيطة مع أزرار تحكم (تعديل / حذف / إضافة) لكل سجل.

## 7. صفحة الدورات (Courses Page)
- شبكة بطاقات لجميع الدورات، يحتوي كل كارت على اسم الدورة، وصف قصير، وزر «اشترك».
- عند فتح دورة محددة تظهر صفحة تفاصيل المحتوى والمخرجات المتوقعة.

## 8. صفحة الدفع (Checkout Page)
- ملخص الاشتراك يوضح الباقة والسعر.
- خيارات الدفع المتاحة: بطاقة، PayPal، Tap.
- زر رئيسي «إتمام الدفع».
- صفحة نجاح منفصلة تتضمن رسالة تهنئة وزر «العودة للوحة التحكم».

## 9. صفحة التقارير
- للطالب: جدول يوضح الحفظ، الغياب، ملاحظات الشيخ.
- لولي الأمر: تقرير مبسّط عن تقدّم الابن.
- للشيخ: إمكانية عرض تقارير مفصلة لكل طالب.

## 10. صفحات إضافية
- **عن المنصة**: عرض الرؤية والرسالة بصورة مختصرة.
- **تواصل معنا**: نموذج (اسم، بريد إلكتروني، رسالة).
- **الأسئلة الشائعة (FAQ)**: قائمة بالأسئلة والإجابات الأكثر تداولًا.
- **سياسة الخصوصية**: نصوص قانونية مبسطة.
- **الملف الشخصي (Profile)**: صورة شخصية، بيانات المستخدم، زر لتعديل المعلومات.

## 11. وظائف عامة
- نظام إشعارات للحصص القادمة والمدفوعات.
- تصميم متجاوب مع جميع الأجهزة (موبايل، تابلت، سطح مكتب).
- دعم الوضعين الفاتح والداكن دون فقدان التناسق البصري.

هذا التصور يمهّد لمرحلة التصميم التفصيلي ويضمن وضوح المتطلبات البصرية والوظيفية قبل البدء بالتنفيذ.
